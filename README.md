### 仓库内容

不是特别专业, 但是应付实验作业还是够用的, 给的分基本都是满分偶尔扣个两三分.

Lab内容如下:

- Lab1: 线性表
- Lab2: 栈
- Lab3: 队列
- Lab4: 二叉树
- Lab5: 图
- Lab6: 查找

没有排序, ~~反正没布置就没写~~

代码实现都在`labx/src`中, `.h` 文件里面包含实验报告中要求的实现, `.cpp`基本就是测试了
